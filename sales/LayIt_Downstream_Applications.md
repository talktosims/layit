# LayIt Laser — Downstream Use Cases & Applications

*Beyond tile: every industry that puts repeating patterns on flat surfaces*

*Last updated: March 15, 2026*

---

## THE CORE INSIGHT

LayIt's proprietary technology does three things no competitor combines:
1. **Computes a repeating geometric pattern** with grout/gap spacing, cuts, and waste
2. **Projects that pattern onto a real surface** with auto-correcting laser alignment
3. **Uses computer vision** to track progress and self-calibrate

This technology stack applies to ANY installation where a repeating unit is placed on a surface in a pattern. Tile is just the first market.

## NEAR-TERM APPLICATIONS (Software update only — same hardware)

### 1. Hardwood & Engineered Flooring
- **Pattern types:** Straight lay, diagonal, herringbone, chevron, basket weave, parquet
- **What changes:** Plank dimensions (3.5" × variable length), stagger rules (random vs fixed offset), expansion gap calculations
- **Market:** $28B US flooring market. Every flooring installer does the same measure-mark-cut workflow
- **LayIt advantage:** Stagger optimization — the app computes plank offsets to minimize waste from random-length boxes. Current installers eyeball this and waste 15-20%. LayIt cuts waste to 5-8%
- **Revenue:** Same hardware, new app tab. Unlocks entire flooring installer market as customers

### 2. Brick & Paver Installation
- **Pattern types:** Running bond, stack bond, herringbone, basket weave, soldier course
- **What changes:** Brick dimensions, mortar joint spacing (3/8" typical), slope/drainage calculations
- **Market:** $8B US masonry/hardscape market
- **LayIt advantage:** Outdoor patios, driveways, walkways — the laser projects the exact pattern including border courses and cuts. Currently done with string lines and chalk
- **Key feature:** Drainage slope visualization — project the pattern tilted to show where water flows

### 3. Ceiling Tile / Drop Ceiling
- **Pattern types:** 2×2 grid, 2×4 grid, diagonal, tegular
- **What changes:** Grid dimensions (24"×24" or 24"×48"), main runner and cross-tee layout, border tile calculations
- **Market:** $3B US ceiling tile market, dominated by commercial construction
- **LayIt advantage:** Project the grid onto the ceiling from below — shows exactly where to install main runners and cross-tees. Currently done with laser levels and string, takes hours for large rooms
- **Pro angle:** Ceiling projection is where keystone correction really shines — projecting UP onto a surface above you

### 4. Stone & Slab Installation
- **Pattern types:** Ashlar, Versailles, pinwheel, modular random
- **What changes:** Multiple tile sizes in one pattern (e.g., Versailles uses 4 different sizes), irregular cut optimization
- **Market:** High-end residential, $5B+ natural stone market
- **LayIt advantage:** Multi-size patterns are extremely hard to lay out manually. The app computes where every size goes and the laser shows it. This is where experienced tile guys still make mistakes

### 5. Vinyl Plank / LVP / LVT
- **Pattern types:** Straight lay, diagonal, herringbone
- **What changes:** Click-lock plank dimensions, no grout, stagger rules
- **Market:** Fastest growing flooring segment — $12B and growing 8% annually
- **LayIt advantage:** LVP is the #1 DIY flooring product. These are the people who rent tools. Perfect fit for the rental program

### 6. Wallpaper & Wall Panels
- **Pattern types:** Straight match, drop match, random match
- **What changes:** Roll width, pattern repeat distance, plumb line projection
- **Market:** $3B US wallpaper market (resurging trend)
- **LayIt advantage:** Project vertical plumb lines and horizontal pattern-match lines on the wall. Shows where each strip goes and where the pattern aligns. Currently done with a level, pencil, and prayer

## MEDIUM-TERM APPLICATIONS (Minor hardware/software modifications)

### 7. Solar Panel Layout
- **Pattern types:** Portrait grid, landscape grid, staggered for hip roofs
- **What changes:** Panel dimensions (typically 65" × 39"), racking spacing, setback requirements, obstruction avoidance (vents, skylights, chimneys)
- **Market:** $30B US residential solar market
- **LayIt advantage:** Project the exact panel layout onto the roof from the ground. Installer sees where every rail, clamp, and panel goes before climbing up. Currently done with chalk lines on the roof in the sun
- **Modification needed:** Longer-range laser (upgrade to 500mW) for outdoor daylight visibility, extended projection distance for roof angles
- **Huge safety angle:** Less time on the roof measuring = fewer falls. Falls are the #1 killer in construction

### 8. Drywall Layout
- **Pattern types:** Horizontal, vertical, staggered seams
- **What changes:** 4×8 or 4×12 sheet dimensions, stud spacing overlay, seam stagger rules
- **Market:** Drywall is in literally every building. $15B US market
- **LayIt advantage:** Project stud locations + sheet layout simultaneously. Shows where to cut, where seams land, where to add blocking for heavy items (TVs, cabinets). Currently: measure, mark, snap chalk line, repeat 200 times per room

### 9. Insulation Layout
- **Pattern types:** Batt placement between studs, rigid board grid
- **What changes:** Stud bay dimensions, R-value optimization, vapor barrier overlap
- **Market:** $10B US insulation market
- **LayIt advantage:** Project cut lines for rigid foam boards, show batt placement in irregular bays. Especially useful for cathedral ceilings and complex geometries

### 10. Cabinet Installation Layout
- **Pattern types:** Linear runs, L-shaped, U-shaped, island placement
- **What changes:** Cabinet widths (12"-48"), filler strips, appliance openings
- **Market:** $20B US cabinet market
- **LayIt advantage:** Project the exact footprint of every cabinet on the wall and floor. Shows where uppers and lowers align, where fillers go, where appliance openings land. Currently: pencil lines on the wall, constant re-measuring

### 11. Commercial Carpet Tile
- **Pattern types:** Monolithic, quarter-turn, ashlar, brick, random
- **What changes:** 24"×24" or 18"×36" tiles, no grout, seam direction
- **Market:** $2B US commercial carpet tile market (Interface, Shaw, Mohawk)
- **LayIt advantage:** Large commercial floors (10,000+ sq ft) with complex furniture layouts and floor box locations. Project the pattern showing exactly where tiles need to be cut around columns, floor boxes, and curved walls

## LONG-TERM / ADJACENT APPLICATIONS

### 12. Concrete Form Layout
- Project form board locations, anchor bolt positions, rebar grid spacing
- Used by concrete contractors for foundations, slabs, walls

### 13. Landscape Design Projection
- Project garden bed layouts, paver patterns, retaining wall courses onto the ground
- Client sees the design in real life before any digging starts

### 14. Stage & Event Setup
- Project dance floor tile layout, booth placement for trade shows, seating grid for events
- Temporary installations that need fast, accurate layout

### 15. Manufacturing / Assembly
- Project part placement patterns on assembly tables
- Pick-and-place guidance for manual assembly operations
- Quality control: verify placement matches spec

### 16. Archaeological / Survey Grid
- Project excavation grids onto dig sites
- Consistent grid spacing for artifact documentation

### 17. Sports Court Line Marking
- Project court lines (basketball, tennis, pickleball, volleyball) onto surfaces before painting
- Currently done with measuring tapes and chalk — error-prone for curved lines
