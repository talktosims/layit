# LayIt Product Roadmap v2
*Updated: March 17, 2026*

## Priority 1 — Ship LayIt (In Progress)
- App Store submission: metadata drafted, screenshots needed
- In-app purchases: $4.99/mo, $39.99/yr
- Barcode DB: 60+ tiles, geometry extraction pipeline built
- New sub-shapes: octdot, arabesque, elonghex, custom vertex renderer
- Remaining: screenshots, App Store Connect setup, IAP products, archive + upload

## Priority 2 — OverLayIt (AR Overlay Feature, v2)
- User takes "before" photo of wall/floor
- App detects surface plane (ARKit plane detection)
- Selected tile pattern rendered at correct scale + perspective on the photo
- Client sees new tile IN their actual space before installation
- Killer upsell for Pro contractors showing clients options
- Could be pre-launch differentiator if built fast enough

## Priority 3 — File 3rd Provisional Patent ($65)
- "Method and System for Automated Extraction of Material Geometry and Tessellation Parameters from Digital Images"
- 22 claims covering: AI geometry extraction, tessellation inference, cross-reference validation, incremental learning
- Broad scope: tiles, pavers, shingles, panels, planks, fabric, solar panels, packaged goods
- Draft complete at: `/Users/Sims/Desktop/layit/patent/LayIt_Patent_GeometryExtraction_Provisional.md`
- File BEFORE app goes public (starts 1-year patentability clock)

## Priority 4 — LayIt Flooring App
- Fork LayIt, simplify to rectangle-only geometry
- No laser needed — software only (layout + stagger + waste optimization + material calc)
- Supports: hardwood, LVP, laminate, carpet tile
- Key features: stagger planning (avoid H-joints), offcut reuse optimization, room shape material estimation
- Bigger market than tile, simpler geometry, pure software margin
- Could ship within a month of LayIt launching

## Priority 5 — Roofing Estimation Software
- Takes contractor's existing drone footage (NOT our drone)
- AI extracts roof geometry from aerial photos
- Generates material estimate: shingle count, ridge cap, flashing, underlayment
- Pricing: $15/report (vs EagleView at $50-100)
- No hardware, no insurance, no truck rolls — pure software
- FAA Part 107 NOT needed if using contractor's own footage

## Priority 7 — LayIt Cut Van (Mobile Fabrication)
- Sprinter van + WAZER waterjet cutter ($7,500) + water/power setup
- Mobile tile fabrication: intricate cuts on-site or shop + delivery
- LayIt generates cut files (SVG/DXF) → WAZER executes
- Services business model, software is the enabler
- Franchise potential: sell van setup + LayIt Pro license + training package
- Capital intensive — do after revenue flowing from software products
